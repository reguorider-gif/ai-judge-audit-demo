Approve, rewrite, rerun, escalate, or stop the loop with a human-readable reason.

Core contract

The product shape is portable JSON.

The homepage tells the story. The JSON packet holds the real operating surface for integrations, review queues, GitHub issues, and future API work.

  • Input: trigger, output, sources, tool logs, judge scores.
  • Audit: claim ledger, evidence status, seat disagreement.
  • Decision: human verdict, safer rewrite, next-run control.

Where it fits

Beside the tools teams already use.

AI Judge is not trying to replace the eval stack. It packages selected failures into something product, research, support, engineering, and compliance reviewers can share.

Ragas, TruLens, DeepEval Turn groundedness and faithfulness failures into claim-level audit packets.
Langfuse, Phoenix, Opik, LangSmith Attach reviewer-readable verdicts and next-run decisions to selected traces.
Support, coding, and research agents Stop recurring automation from repeating one polished but unsafe answer.
